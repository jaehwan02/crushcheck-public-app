# Public Demo Usage Checklist

## 빠른 점검 (2분)

1. 앱 시작 후 홈 화면에서 보유 토큰과 받은/보낸 마음 카드를 확인합니다.
2. 보내기 탭에서 친구를 검색하고 메시지를 입력해 전송을 완료합니다.
3. 전송 완료 화면에서 "상세 보기"로 이동해 신호 상세를 확인합니다.
4. 설정 탭에서 API 모드(정상/빈 응답/오류)를 전환해 예외 화면을 확인합니다.
5. 오류 복구가 필요하면 데이터 초기화 버튼으로 초기 상태를 복구합니다.

## 상태 케이스

- 정상: 홈/상세/보내기 모두 성공 응답
- 빈 응답: 홈 피드, 친구 목록이 빈 상태로 표시
- 오류: 재시도 동작과 에러 상태 메시지 동작 확인
